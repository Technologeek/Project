# Project
Virtual Studio Technology for all Digital Audio Workstations .Amplification of signals to create Robotic Sounds of the given MIDI signal.
This is a free opensource VSTI for all the Digital Audio workstations .The final functioning model of the VST is expected
to module any MIDI signals and amplify them to create various robotic sounds which could be used by music producers or any one who 
needs application of robotic sounds of their MIDI samples.
This is developed using C++ and JUCE library .
